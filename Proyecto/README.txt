Proyecto de Analisis de poblacion

Objetivo: Analisar los datos de la creciente poblacion entre mutiples paises, filtrando datos como solamente paises de habla hispana.
Utilizando carga de archivos CSV y Pandas

Tareas que ejecuta el codigo: 
* Graficacion de Paises
* Creacion de Tablas de datos descargados desde github

Usos del Codigo:
* Facilitar el analisis de datos de poblacion por medio de python

REQUERIMIENTOS:
Es necesario hacer descargas de github desde repositorios:
https://github.com/jbagnato/machine-learning/blob/master/Ejercicio_EDA.ipynb
