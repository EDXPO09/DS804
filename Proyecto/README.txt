Proyecto de Automatizacion

Objetivo: Facilitar la creacion de carpetas para almacenar documentos. En este caso se decide crear carpetas
de los meses del año 2023 (Esto puede ser cambiado dentro del propio codigo).

Tareas que ejecuta el codigo: 
* Creacion de 12 carpetas con los meses del año 2023 (Puede ser modificado)
* Creacion de Carpetas acorde al mes del año.

Usos del Codigo:
* Facilitar la organizacion de Documentos Excel a la hora de exportar datos, facilitando el almacenamiento
* Puede agregarse codigo Extra para ser capaz de Extraer Datos de Excel y organizarlos segun los dias asignados
* Resulta Conveniente de utilizar para las tareas tediosas
